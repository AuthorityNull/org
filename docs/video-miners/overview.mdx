# Overview

Welcome! The pages that follow contain comprehensive documentation of the
Livepeer ecosystem. The documentation is split into two parts: Livepeer for
video developers and Livepeer for video miners. If you are new to Livepeer and
interested in how it works, you might want to check out the Protocol Overview
[Core Concepts](https://nextjs.org/)
